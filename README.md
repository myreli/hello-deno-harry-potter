## Requirements
```shell
deno 1.0.0
v8 8.4.300
typescript 3.9.2
```

## Up & Running

```shell
# clone this branch
git clone <url> --branch hello-web-apis --single-branch
# change directory 
cd hello-deno-harry-potter
# run
deno run mod.ts
```
