## Requisitos

```shell
deno 1.0.0
v8 8.4.300
typescript 3.9.2
```

## Up & Running

```shell
# clone this repo
git clone https://github.com/myreli/hello-deno-harry-potter.git
# change directory
cd hello-deno-harry-potter
# run
deno run --allow-net --allow-write mod.ts hermione
# test 
deno test
```
